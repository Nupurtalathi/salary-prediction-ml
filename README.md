# Salary Prediction System (ML Project)

End-to-end machine learning system that predicts salary ranges from job postings using feature engineering, classification models (Random Forest, Decision Trees, CatBoost), and a deployed Streamlit application.
Salary Prediction System (ML Project)

This project builds an end-to-end machine learning pipeline to predict salary ranges based on real-world job posting data (30,000+ records across 12,000+ companies).

The system focuses on handling noisy, real-world datasets and transforming them into meaningful features for accurate prediction.

🔍 Key Highlights
1)Performed data cleaning and preprocessing on large-scale job data
2)Engineered features such as job title normalization and geographic tier classification
3)Identified and resolved data leakage issues in the pipeline
4)Converted salary prediction from regression to range-based classification
5)Built and compared multiple models: Random Forest, Decision Trees, and CatBoost
6)Designed a dual-model approach for scenarios with and without company information
7)Deployed an interactive web app using Streamlit

⚙️ Tech Stack
Python · Pandas · Scikit-learn · CatBoost · Random Forest · Decision Trees · Streamlit

📊 Results
Improved prediction stability using classification over regression
Achieved strong performance across multiple model comparisons
Delivered real-time predictions via web interface
